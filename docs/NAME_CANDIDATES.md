# Name Candidates — product name for Yingzao

> The product is the **evidence-based reconstruction engine** (enter a historic place → research → rebuild it with provenance). Current codename **Yingzao** (营造 = "to build / construct / create") reads as China-only. We want a **techy, era/culture-agnostic, ownable** product name — ideally evoking *build / create / re-create* (like 营造) and/or *ground-truth / provenance* (the differentiator). ★ = top picks.

## Direction A — meaning = "to build / create / re-create" (like 营造)
- **Tekton** ★ — Greek *"builder / maker"*; the root of **architect** (archi-tekton). Best meaning-match; short, techy, ownable.
- **Fabrica** ★ — Latin *"the making / workshop"*; the historic cathedral-building office (*Veneranda Fabbrica del Duomo*) — strong heritage resonance.
- **Edify / Edifice** — Latin *aedificare* "to build"; *edify* also = to enlighten/instruct.
- **Recreo / Recrea** — Latin *"I re-create / restore"* — the most literal "recreation."
- **Zōkei** (造形) — Japanese *"formative creation"*; shares the **造** ("make/build") character with 营造.
- **Zukuri** (造り) — Japanese *"making"* (as in *monozukuri*); also 造.
- **Struxa / Construo** — Latin *struere* "to construct."
- **Erige** — Latin *erigere* "to raise / erect."
- **Restituo** — Latin *"to restore / rebuild."*
- **Maquette** — French *"scale model"* (also nods to Notre-Dame).
- **Poiesis** — Greek *"making"* (root of *poetry*; cf. *autopoiesis*).
- **Demiurge** — *"the craftsman-maker."*
- **Bau / Bauen** — German *"build"* (the *Bauhaus* root).

## Direction B — techy + on-thesis (provenance / ground-truth / verification)
- **Groundtruth / GroundTruth** ★ — the ML term for verified reference data; *ground* (heritage) + *truth* (provenance/verifier). AI-native, generic.
- **Keystone** ★ — the stone that locks an arch / "the key piece"; reads as a platform name.
- **Provenant** ★ — from *provenance*, the core differentiator; sounds like a product, ownable.
- **Reify** — the programmer's *"make the abstract concrete"*; short, techy.
- **Datum** — a survey reference point + *data*.
- **Quarry** — where material is *sourced* (a provenance pun).
- **Attest · Veritas · Verity** — truth / attestation.
- **Ledger · Vellum** — the record / the manuscript source.
- **Truss · Scaffold · Lattice · Substrate · Cornerstone** — structural, generic.
- **Plumb** — the plumb-line = trueness / accuracy.
- **Cairn · Strata · Tessera** — heritage / layered / mosaic.

## Top 4 overall
**Tekton** (meaning) · **Groundtruth** (thesis) · **Fabrica** (heritage) · **Provenant** (ownable).

- A **naming-evaluation agent goal** (pressure-test + availability check + recommendation) is ready to run.
